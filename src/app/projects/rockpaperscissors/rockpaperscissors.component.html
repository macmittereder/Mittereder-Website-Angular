Rock, Paper, Scissors

This app was developed junior year of high school for Android using eclipse. Theidea of this app was a proof of concept to develop my first app and publish it on the play store. During a java class we developed a rock, paper, scissors game with command line style input so I ported it to be more interactive in an app setting.

This view is the main menu that you are greeted with when opening the app."

This is the single player view where you can make your selection off the bottom 3 choices."


The player has made the choice of the scissors and the computer has chosen rock, so the person would lose in this case."

This time the player chose rock and the computer had chosen scissors so the person would win"


This is the stats screen where all the stats are stored.

This view is the prefences menu where you can change the font in game. You have the option to keep track of your stats and you can also clear stats data.
